# BLS Data Downloads

Bulk downloaded from U.S. Bureau of Labor Statistics using the [BLSloadR](https://github.com/schmidtDETR/BLSloadR) R package.

## Datasets

| Code | Dataset | File |
|------|---------|------|
| ce | Current Employment Statistics (National) | bls_ce_data.csv |
| cu | Consumer Price Index | bls_cu_data.csv |
| jt | Job Openings (JOLTS) | bls_jt_data.csv |
| pr | Producer Price Index | bls_pr_data.csv |
| is | Import/Export Price Indexes | bls_is_data.csv |
| ci | Employment Cost Index | bls_ci_data.csv |
| sm | State and Area Employment | bls_sm_data.csv |
| sa | State and Area Employment | bls_sa_data.csv |
| la | Local Area Unemployment Statistics | bls_la_data.csv |
| oe | Occupational Employment Statistics | bls_oe_data.csv |
| mp | Major Sector Productivity | bls_mp_data.csv |
| qcew | Quarterly Census of Employment and Wages | bls_qcew_data.csv |

## Source

- BLS Bulk Download: https://download.bls.gov/pub/time.series/
- BLSloadR Package: https://github.com/schmidtDETR/BLSloadR

## GitHub

- Repository: https://github.com/luqmaan-cmd/bls-economicdata-pipeline

## Download Script

See `bls_bulk_download.R` for the R script used to download these datasets.
